Acerca de esta Web

Antes que nada, me es muy grato saber que alguien se esta tomando el tiempo de leer estas líneas introductorias. Basicamente, este sitio está dedicado principalmente a la Seguridad de la Información con énfasis en la criptografía aplicada, fundamentos y complejidad.

Intentaré, en la medida de lo posible, la mayor rigurosidad, sin embargo no puedo garantizarla.

Intereses de investigación:

  • Asymptotic Analysis \(\lambda\): Characterizing security guarantees as functions of parameter growth.
  • Classical Cryptography: C/Rust treatment of fundamental primitives and constructions.
  • Complexity-Theoretic Foundations: assumptions, and provable security.
  • Algorithmic Cryptography:Efficient implementations with formal correctness guarantees.

Decálogo:

  • Code should illuminate theory.
  • Every implementation is paired with rigorous documentation explaining the underlying mathematical constructs.
  • Repositories in this organization serve as pedagogical implementations for engineering and computer science students.
